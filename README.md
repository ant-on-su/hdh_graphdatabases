### Neo4j sandbox

- ga naar https://sandbox.neo4j.com
- maak een account
- start Graph Data Science project
- klik op grijze carret om connection details te zien
- klik op Connect via drivers
- Kies voor Python
- Kopieer de code naar de de notebook `hdh.ipynb`
